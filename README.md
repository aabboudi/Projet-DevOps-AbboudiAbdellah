Projet-DevOps-Abdellah-Abboudi

Auteur: Abdellah Abboudi 
5IIR G1

L'objectif principal est de valider les compétences pratiques sur le cycle de vie de développement logiciel. Ce projet démontre l'utilisation de Git, l'automatisation via GitHub Actions et la mise en place d'une pipeline CI/CD avec Jenkins et Docker.
